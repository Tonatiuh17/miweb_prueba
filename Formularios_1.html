Formularios sencillos (Teoría)

Para usa formularios usamos la etiqueta "form", entre los pico paréntesis, junto con las siguientes etiquetas y atributos:

Etiquetas
  1. input: Define un control de entrada
  2. textarea: Define un contrl de entrada multilínea
  3. label: Define una etiqueta para un elemento de entrada
  4. fieldset: Agrupa el elemento relacionado en un formulacio
  5. legend: Define un título para un elemento "fieldset".
  6. select: Define una lista desplegable
  7. optgroup: Define un grupo de opciones relacionadas en una lista despregabñe
  8. option: Define una opción en una lista despregable.
  9. button: Define un botón en el que se puede hacer clic
  10. datalist: Especifica una lista de opciones predefinidas para el control de entrada.
  11. kaygen: Define un campo generador de pares de claves para formularios.
  12. output: Define el resultado de un cálculo
Atributos
  1. action: URL. Dirección URL del back-end donde se enviará la información del formulario.
  2. method. get I post. Método HTTP de envío. GET a través de URL, POST para envío extenso.
  3. name: nombre. Nombre del formulario. Útil para procesar posteriormente.
  4. target: destino. Nombre del lugar donde se abrirá el formiulacio. \_blank para nueva pestaña
  5. enctype: tipo. Codificación para el envío del formulario. Imporatnte para envío de archivos.
  6. accpt-ch arset. codificación. Fuerza a utilizar una codificación en los parámetros de texto del fromulario.
  7. autocompl ete. on|off. Activa o desactiva el autocompletado para todos lo campos del formualrios.
  8. novalidate: COn este atributo presente, el formulario obvia la validación HTML5
  9. type: tipo de campo. Indica el tipo de campo del que se trata.
